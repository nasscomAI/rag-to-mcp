# Platforms

This list includes platforms for AI, data science, machine learning, and vector databases.

* [Kaggle](https://www.kaggle.com/) - A platform for data science competitions, datasets, and notebooks.
* [Hugging Face](https://huggingface.co/) - The platform where the machine learning community collaborates on models, datasets, and applications.
* [ChromaDB](https://www.trychroma.com/) - Open-source embedding database (used in this workshop).
* [Pinecone](https://www.pinecone.io/) - Managed vector database for production RAG systems.
* [Weaviate](https://weaviate.io/) - Open-source vector search engine.
* [Qdrant](https://qdrant.tech/) - High-performance vector similarity search.
* [LangSmith](https://smith.langchain.com/) - Platform for debugging, testing, and monitoring LLM applications.
